# /brainstorm — Deprecated

This command has been replaced. Run the same quick 3-perspective gut check via:

```
/council --brainstorm <idea>
```

For structured divergent ideation (questions before solutions), invoke the
`superpowers:brainstorming` skill instead.

If the user invoked `/brainstorm <idea>`, run `/council --brainstorm <idea>`
on their behalf now and mention the new entry point once.
