# Convention References

Battle-tested conventions for Python, TypeScript, and Terraform projects. Extracted from production use and generalized for reuse.

## Quick Start

1. **New project?** Use the scaffolding commands: `/new-python`, `/new-typescript`, `/new-terraform`
2. **Reviewing code?** Reference the language convention file for the relevant stack
3. **Setting up project infrastructure?** See [project-claude-md.md](project-claude-md.md) and [codemap.md](codemap.md)
4. **Need a PRD?** See [prd.md](prd.md) for the template
