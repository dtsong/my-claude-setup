# Findings Ledger

This file is an append-only knowledge base for cross-engagement trend tracking.
It is intentionally minimal and may be curated by engineers.

## Entries

### Finding Template

```markdown
## [FINDING-{YYYY}-{NNN}] — {Title}
- Date: {YYYY-MM-DD}
- SoC Family: {family}
- Technology Domain: {domain}
- Standards: {standards list or none}
- Severity: {critical|high|medium|low|informational}
- Confidence: {GROUNDED|INFERRED|SPECULATIVE|ABSENT}
- Finding: {Layer 2 business risk statement}
- Resolution: {Layer 3 recommended action}
- Reusability: {High|Medium|Low}
- Related: [{upstream entity IDs, TM-IDs, CM-IDs, VC-IDs}]
```

### Decision Template

```markdown
## [DECISION-{YYYY}-{NNN}] — {Title}
- Date: {YYYY-MM-DD}
- SoC Family: {family}
- Technology Domain: {domain}
- Standards: {standards list or none}
- Decision: {what was decided}
- Rationale: {why}
- Tradeoffs: {what was accepted}
- Related: [{upstream entity IDs, TM-IDs, CM-IDs, VC-IDs}]
```
